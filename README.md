# Swift Express Logistics

Swift Express Logistics is a modern web-based logistics and parcel tracking system designed to simulate real-world courier operations.

## 🚀 Features
- Real-time shipment tracking (demo system)
- Admin dashboard for managing tracking IDs
- User login interface
- AI chatbot support assistant
- Responsive design for mobile and desktop
- Floating contact buttons (Telegram & Email)

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
